> A curated list of awesome projects, tools, and resources in the Etherlink ecosystem

**Etherlink** is an EVM-compatible, non-custodial Layer 2 blockchain powered by Tezos Smart Rollup technology, offering high-performance scaling, low transaction costs, and seamless interoperability with Ethereum. This comprehensive list showcases the rapidly growing ecosystem of projects, developer tools, DeFi protocols, gaming platforms, and infrastructure that make Etherlink a leading destination for blockchain innovation.

## Contributing

Contributions are welcome! Please read our [contributing guidelines](CONTRIBUTING.md) before submitting pull requests.

### How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-project`)
3. Add your project with proper formatting
4. Ensure all links work and descriptions are accurate
5. Submit a pull request

### Project Requirements

- Projects must be actively maintained and operational on Etherlink
- Must have a working website or documentation
- Must provide clear value to the Etherlink ecosystem
- Should be open source where possible
- Must be genuinely integrated with Etherlink (not just planned)

### Submission Format

```markdown
- **[Project Name](https://example.com)** - Brief, clear description of the project
```
